# DES Project 2
by Antonio Gutierrez


# Project Overview
This project aims to portray my definition of community through an interactive webpage using only words and motion to guide the user through it. I chose this excerpt from this song by Nujabes not just because I really like the music, but because to me community means a group of people that can take you away from hurt and despair. Whether its your family, friends, colleagues, or others, your community should always have your back and help you achieve "something that is greater". I think the pandemic limited my way to find community in person and in ways that are still present today, but instead I was able to make strong connections with people online. Trying to forge new communities as I get older and transition more into adulthood scares me sometimes, but I always have my family and friends no matter what.


# Technical Overview

Coded in HTML and CSS using Visual Studio Code. Uses a lot of keyframe animations to create motion that guides the user from page to page.
